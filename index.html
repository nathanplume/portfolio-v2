Grocery Buddy

Grocery Buddy is a todo list style application utilising HTML, CSS and vanilla Javascript. You can add, delete and edit grocery items to your list. A notification will let you know if it has been successful or not. Items are also added to local storage so they will not disappear on a page refresh.
